```
str_replace(find,replace,string,count)
```
| 参数 | 描述 |
| --- | --- |
| find | 必要。规定要在string中查找的值 |
| replace | 必要。将查找到的值替换为replace中的值 |
| string | 必要。原字符串 |
| count | 非必要。对替换的字符串进行计数
